# 使用方式

## 1，安装插件

> 下载后解压，浏览器进入插件管理界面，开启【开发者模式】，点击【加载解压缩的扩展】，导入插件。

## 2，打开一个twitch直播，再打开一个YouTube直播，就可以在twitch端同时显示twitch和YouTube弹幕了

> YouTube聊天需要保持跟随状态。

> YouTube聊天点击右上角三个点，跳出聊天，就可以关闭原本的YouTube直播页面了。

> 若出现问题，可以先刷新一下twitch，确保twitch弹幕正常显示，再刷新一下YouTube尝试解决。
